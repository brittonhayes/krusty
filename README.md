> Krusty looks through directories for files you haven't modified in a while

## Installation

How to install krusty

```shell
cargo install --git https://github.com/brittonhayes/krusty
```

## Usage

How to use krusty

```shell

# View help command
krusty --help

# Search for krusty files in a directory
krusty crawl ./
```
